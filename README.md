# dz_20.11
Создайте программу для управления списком дел, используя объектно-
ориентированное программирование (ООП). Программа должна позволять:
1. Добавлять задачи в список.
2. Помечать задачи как выполненные.
3. Просматривать текущий список задач.
4. Сохранять данные в файл и загружать их при следующем запуске программы.

Указания:
1. Реализуйте два класса:

1) Task
Поля:
- title — название задачи.
- is_done — статус задачи (по умолчанию: "Не выполнено").
Методы:
- mark_done() — помечает задачу как выполненную.

2) TaskManager:
Поля:

- tasks — список задач (загружается из файла при запуске).

Методы:
- add_task(title) — добавляет новую задачу.
- mark_task_done(title) — помечает задачу как выполненную.
- show_tasks() — отображает список задач с их статусами.
- save_to_file() — сохраняет текущий список задач в файл.
- load_from_file() — загружает список задач из файла.

2. Программа должна работать в интерактивном режиме с командами:
- add — добавить новую задачу.
- done — отметить задачу как выполненную.
- show — вывести список задач.
- exit — сохранить данные в файл и завершить программу.


3. Файл для хранения задач должен быть в формате JSON.


Ожидаемый результат:

1. При добавлении задачи:

Введите команду: add
1. Введите название задачи: Убраться в комнате
2. Задача "Убраться в комнате" добавлена.

2. При отметке задачи как выполненной:

Введите команду: done
1. Введите название задачи: Убраться в комнате
2. Задача "Убраться в комнате" отмечена как выполненная.

3. При выводе списка задач:

Введите команду: show
1. Убраться в комнате - Выполнено
2. Сделать уроки - Не выполнено
4. При выходе из программы:

Введите команду: exit
Данные сохранены. Программа завершена.

При следующем запуске программы задачи загружаются из файла:

Введите команду: show
1. Убраться в комнате - Выполнено
2. Сделать уроки - Не выполнено
